            Visibility Property
            The visibility property controls whether an element is visible or hidden in the document.
            Important Note:
            •	visibility: hidden;: The element is invisible, but the space it occupies is still reserved.
            •	display: none;: The element is removed from the document flow, and no space is reserved for it.
            Syntax:
            visibility: hidden;


this is relative units of css


    Percentage (%):( in short it take space of the parents’ size.)
    Defines size relative to the parent or other properties.
    Parent and child example:
    div style="width: 100%; height: 200px; background: lightgray;"
    h1 style="width: 50%; height: 50%; background: lightblue;" 
    ________________________________________ 

    Em:( in short it multiplies to the parents’ size but it multiplies to font-size of itself and with parents’ if width is taken in “em”.)
    Relative to the element's font size.
    Example with padding and font-size: 

    div style="font-size: 20px; padding: 1em; border: 1px solid black;"
    p style="font-size: 0.5em;" Font size is half of parent /p
   
    ________________________________________
        
    Rem:(it multiplies with default value of html text root without any style.)
    Relative to the root font size.
    ________________________________________
    vh and vw
    Relative to viewport size.
    Example:
    div style="width: 50vw; height: 50vh; background: lightgreen;"
    50% viewport width and height
